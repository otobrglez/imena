# Slovenian names

Tools and collection of Slovenian names, sorted by sex.

## Installation & development

    mkvirtualenv \
      --no-site-packages \
      --python=/usr/local/Cellar/python3/3.4.3_2/bin/python3 imena
    workon imena
    pip install --upgrade -r requirements.txt

## Compiling list

    make
